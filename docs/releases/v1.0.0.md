## Included Operator Commands

Start locally:

```powershell
py -3.10 -m pip install -r requirements.txt
py -3.10 -m uvicorn backend.collaboration.app:app --host 127.0.0.1 --port 8080
```

Run the release gate:

```powershell
py -3.10 scripts/collaboration-release/readiness_check.py --with-pytest --json
```

Run v5.2 acceptance:

```powershell
py -3.10 scripts/collaboration-release/v52_acceptance.py --json
```

Run self-hosted relay smoke:

```powershell
py -3.10 scripts/collaboration-relay/self_hosted_smoke.py `
  --base-url http://localhost:8080 `
  --relay-mode self_hosted `
  --relay-url local://memory `
  --json
```

Run with Docker Compose:

```powershell
docker compose up --build
```

## Known Boundaries

`v1.0.0` is a local-first / self-hosted MVP release. It does not implement:

- billing or payment enforcement
- user accounts, authentication, tenancy, SSO, or LDAP
- a production SaaS relay
- persistent SQLite/Redis storage
- automatic source-code synchronization or merge resolution

Relay paths remain metadata-only: room state, lock state, event metadata, and
audit metadata. Source code contents must not be transmitted through relay
paths.
